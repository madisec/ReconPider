# ReconPider

An automation Python tool for subdomain enumeration and other features. ( For university presentation )

## Install

```bash
git clone https://github.com/madisec/ReconPider.git
cd ReconPider
chmod +x setup.sh
./setup.sh
python3 reconpider.py -h
```
